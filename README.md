# 🎬 Netflix UI Clone

Uma réplica responsiva da interface principal da Netflix desenvolvida para fins de estudo e aprimoramento de habilidades em desenvolvimento web front-end estático.

> 🚀 Projeto focado na fidelidade de design (UI) e organização estrutural moderna utilizando boas práticas de arquitetura CSS.

---

## ✨ Demonstração

Visualize o projeto publicado e funcionando online através do link:
🔗 [Clique aqui para acessar o projeto](https://github.com/Francacommit/netflix-homepage-replica/new/main?filename=README.md)

---

## 🛠️ Tecnologias Utilizadas

*   **HTML5:** Estruturação semântica da interface de streaming.
*   **CSS3:** Estilização avançada com variáveis globais, Flexbox para alinhamentos e Grid Layout para a distribuição dos cartões.
*   **Google Fonts:** Tipografia *Poppins* integrada de forma otimizada para garantir alta performance.

---

## 📂 Arquitetura de Pastas do Projeto

O código foi organizado de forma modular para facilitar a manutenção e escalabilidade do CSS:
```text
├── style/
│   ├── index.css  # Estilos globais, resets e componentes principais
│   └── media.css  # Regras de responsividade (@media queries) para telas menores
├── index.html     # Estrutura principal da página
└── README.md      # Documentação do projeto
```

---

## 📈 Conceitos Praticados

*   **Design Responsivo:** Uso de regras no `media.css` para garantir que a interface fique impecável desde celulares até telas de desktop.
*   **Conventional Commits:** Histórico de alterações padronizado (ex: `feat(style): ...`) para manter o repositório alinhado aos padrões do mercado de tecnologia.
*   **Gerenciamento de Fluxo:** Aplicação de propriedades de controle de encolhimento e dimensionamento de componentes para evitar quebras visuais em resoluções menores.

1. Clone o repositório para a sua máquina:

3. Clique duas vezes no arquivo `index.html` para abrir diretamente no seu navegador.
